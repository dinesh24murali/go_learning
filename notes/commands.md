# Commands:

Your can also pass a file name, if you don't pass a file name it will build all the projects in the directory
```
go build <file name>
```
For multiple files
```
go build <file name> <file name>
```

```
go run <file name>
```

Format code in the current directory
```
go fmt  <file name>
```

Handle dependencies
```
go install 
go get
```

```
go test
```
